CO2 Concentration Mapping Mobile Robot

The objective was to create a cost-effective mobile robotic solution for Property Management firms, enhancing the efficiency of indoor CO2 concentration monitoring compared to existing market offerings. Constructed a comprehensive business case, assessed the economic life cycle, and conducted a thorough market analysis for the mobile robot.

Software

    Created a ROS2 stack to control the robot while periodically taking Co2 measuremnts. This invlolved using RViz to derive the location of the robot within the room, then creating a Python algorithm to interpolate the concentration data on a 2D plane.

Hardware

    Used a small Lynxmotion differential drive robot to take data measurements at different spots in the room. Additional task specific hardware included:

    • Raspberry Pi 4
    • RPLiDAR SLAM LiDAR sensor
    • Co2 concentration sensor
    • Arduino Uno
    • Logitec game controller
